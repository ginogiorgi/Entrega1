Bienvenido a la entrega intermedia del PROYECTO FINAL:

Importante:
1- Cambiar la ruta de acceso a los templates desde el archivo settings.py para tener acceso a las vistas.

Funcionalidades:
Una vez corriendo el servidor django va a haber una pagina de inicio con poca funcionalidad y opciones. En la nav bar tendremos una seccion de series y peliculas, otra de mangas y comics, una de noticias(proximamente va a tener una vista) y la de ingresar usuario.

Series y peliculas: te lleva a una vista de formularios para realizar la carga de alguna serie o pelicula GEEK y abajo de ese formulario tambien te permite realizar la busqueda por nombre similar.

Manga y comics: te lleva a una vista de formularios para realizar la carga de algun comic o manga. No inclui la busqueda debido a la consigna pero proximamente se hara.

Ingreso de usuario: una especie de login que proximamente se convertira en un registro de usuario para realizar comentarios y dar cierto feedback.
